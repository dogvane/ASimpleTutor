## 约定

- MVP 仅面向“单本书/单个目录”学习场景，不做多项目管理/切换。
- MVP 不做学习闭环（进度、错题本、报告、章节回顾/考核等）。
- 文档内出现的“Agent”指用于：知识体系建立、内容总结、习题生成、练习反馈 的智能组件（可由一个或多个子 Agent/步骤组成）。
