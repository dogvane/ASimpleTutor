# 知识点浏览与搜索（MVP）

## 1. 功能目标

- 在主界面展示知识点列表
- 支持按章节/重要性组织
- 支持搜索框输入关键词定位知识点
- 点击知识点进入学习视图

## 2. 信息架构（IA）

- 左侧（或顶部）导航：
  - 章节树（可折叠）
  - 或“重要性排序”的扁平列表（MVP 可先做扁平）
- 主内容区：
  - 选中 KP 的学习内容与原文对照

## 3. 列表展示策略

- 默认排序：章节树顺序
- 可切换：按重要性降序
- 每个条目展示：
  - KP 标题
  - 所属章节路径（精简展示）
  - 可选：重要性/标签

## 4. 搜索策略（MVP）

- 输入：用户 query
- 输出：KP 列表（匹配标题/别名/证据片段关键词）

### 4.1 匹配规则

- 标题/别名命中：高权重
- 章节路径命中：中权重
- 证据片段命中：低权重

### 4.2 结果呈现

- 展示 Top N（例如 20）
- 高亮命中片段（标题或章节路径）
- 点击结果进入对应 KP 学习页

## 5. 空状态与异常

- 无知识点：提示检查目录/文档格式，提供“重新扫描”按钮
- 搜索无结果：提示换关键词，展示相近章节
