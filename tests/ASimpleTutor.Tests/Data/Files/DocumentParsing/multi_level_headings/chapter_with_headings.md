# 第一章：入门指南

## 1.1 什么是 ASimpleTutor

ASimpleTutor 是一个基于书籍 Markdown 文档的智能学习系统。它能够帮助用户从文档中提取知识点，并提供学习辅助功能。

该系统的主要特点包括：
- **自动化扫描**：自动发现并解析指定目录下的 Markdown 文件
- **智能提取**：使用 AI Agent 自动识别和提取知识点
- **原文对照**：每个知识点都可以追溯到原始文档位置

## 1.2 快速开始

### 环境要求

在开始使用之前，请确保您的环境满足以下要求：
- .NET 10.0 或更高版本
- 至少 4GB 可用内存
- 100MB 磁盘空间

### 安装步骤

1. 下载最新版本的安装包
2. 解压到指定目录
3. 运行 `setup.exe` 进行安装
4. 按照向导完成配置

```bash
# 示例命令
dotnet run --project src/ASimpleTutor.Api
```

## 1.3 基本概念

在学习使用本系统之前，您需要了解以下基本概念：

- **Book Root**：指定要扫描的书籍根目录
- **Knowledge Point**：从文档中提取的知识单元
- **Snippet**：原文片段引用

> **注意**：初次使用建议从默认书籍开始学习。
